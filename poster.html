🚀 免费部署教程（3种方式，任选其一）

方式一：Vercel（推荐 · 最简单）
  1. 访问 vercel.com，用 GitHub 账号登录（免费注册）
  2. 把 static-deploy 文件夹上传到 GitHub 仓库
  3. 在 Vercel 中导入该仓库，一键部署
  4. 获得网址如：https://你的项目.vercel.app
方式二：GitHub Pages（免费 · 永久有效）
  1. 注册 GitHub 账号
  2. 创建仓库，上传 static-deploy 文件夹里的文件
  3. Settings → Pages → 选择分支 → Save
  4. 获得网址如：https://你的用户名.github.io/仓库名
方式三：Netlify（免费 · 拖拽部署）
  1. 访问 netlify.com，注册账号
  2. 把 static-deploy 文件夹直接拖到 Netlify 页面
  3. 自动部署完成，获得网址

💡 部署获得网址后，回到本页面输入网址 → 生成QR码 → 保存海报图片 → 发朋友圈！
